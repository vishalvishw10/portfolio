# Vishal Vishwakarma — Portfolio

Personal portfolio for **Vishal Vishwakarma** — Offensive Security Consultant &
Security Analyst specialising in **AI / LLM penetration testing** and **cloud security**.

Live at **https://rootxvishal.com**

## Stack
Plain, framework-free **HTML / CSS / JS** — no build step.

```
index.html        # markup + all sections
css/styles.css    # hand-authored design system
js/main.js        # nav, scroll reveal, typing effect, checklist, carousel
CNAME             # custom domain for GitHub Pages
```

## Local preview
Open `index.html` directly, or serve the folder:

```bash
npx serve
# or
python -m http.server 8099
```

## Deploy
Hosted on **GitHub Pages**. Any push to `main` republishes the site within ~1 minute.
